# vue-fruit-mart

Simple web app to simulate online fruit store, making use of:


- Vue.js
- Bulma.io

## Concepts

- Data binding, with **v-bind** directive
- List rendering, with **v-for** directive
- Click handling (**v-on:click**) and computed properties
